== Description ==
This plugin adds a list of categories or/and pages to the footer of a page using the storefront theme.

== Installation ==
Install from WordPress plugin gallery or download the zip from https://frametagmedia.com.au/pluginpages/woo-categories-list-in-footer/ and install manually via the plugin upload in WordPress.
